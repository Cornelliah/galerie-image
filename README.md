Une **galerie d'images** dynamique, responsive et modulaire, construite avec **React.js**. Ce projet utilise la puissance des composants React pour gérer l'état de la galerie, l'affichage des images et la visionneuse modale de manière efficace.

---

##  Fonctionnalités

* **Composants Modulaires :** Structure claire basée sur des composants React.
* **Grille Responsive :** Mise en page flexible qui s'adapte à toutes les résolutions d'écran.
* **Visionneuse Modale (Lightbox) :** Affichage plein écran d'une image avec gestion de l'état via React.
* **Gestion de l'État :** Utilisation de `useState`  pour gérer l'image sélectionnée et la visibilité de la modale.

---

##  Démarrage Rapide

Ces instructions vous guideront pour configurer et lancer le projet de développement sur votre machine locale.

### Prérequis

Vous devez avoir **Node.js** et **npm** (ou **Yarn**) installés sur votre système.

* 

[Image of Node.js logo]


### Installation

1.  **Cloner le dépôt** :
    ```bash
    git clone [https://github.com/Cornelliah/galerie-image.git](https://github.com/Cornelliah/galerie-image.git)
    ```

2.  **Naviguer** dans le répertoire du projet :
    ```bash
    cd galerie-image-react
    ```

3.  **Installer les dépendances** :
    ```bash
    npm install
    # ou si vous utilisez Yarn
    # yarn install
    ```

4.  **Lancer l'application** en mode développement :
    ```bash
    npm start
    # ou
    # yarn start
    ```

L'application devrait s'ouvrir automatiquement dans votre navigateur
